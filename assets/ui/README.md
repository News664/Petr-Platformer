# UI

Icons, panel frames, 9-patches, ledger page dressing.

- **Sizes:** icons 128×128 (shipped 64), panel frames as 9-patch PNGs
  (corner ≥24 px), cursor/markers 32×32.
- **Naming:** `icon_<name>.png`, `panel_<name>.png`, `marker_<name>.png`.
- **Key pieces:** chisel-light mote, grace hourglass, ledger book, map
  cell frames, Waystone marker, anchored-chain marker, dialogue panel
  frame, the Amé HUD figure frame.
