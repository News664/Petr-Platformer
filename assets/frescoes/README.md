# Frescoes — rescue vignettes & truth panels

Stylized single-panel images: each rescued NPC's "frozen moment"
vignette (~24) and the 6 Fresco Hall truth panels. The deliberately
flat, iconic style is thematically right (stone imagery) and hides AI's
multi-character consistency weakness — this is the workhorse format for
any scene with several figures.

- **Size:** generate 1024×1024, ship at 1024×1024.
- **Naming:** `fresco_<char>.png` for vignettes,
  `truth_<n>_<slug>.png` for the Fresco Hall.

## Prompt template

> {STYLE_TAG}, carved stone fresco panel in shallow relief, muted
> mineral pigments, gold leaf accents, stained-glass composition,
> {scene_desc}, ornamental stone border, flat iconic perspective

One strong shared style makes all panels read as a set; generate the
whole batch with identical style wording.
