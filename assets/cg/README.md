# Event CGs

Full-scene illustrations for major story beats only (~14–18 total —
keep this list disciplined; sprite cutscenes are the default).

- **Size:** generate 1536×1024 or 1792×1024, ship at 1920×1080
  (upscale/outpaint as needed), PNG or high-quality WebP.
- **Naming:** `cg_<beat>_<nn>.png` — e.g. `cg_opening_01.png`,
  `cg_ending_true_03.png`.
- **Planned list:** opening calamity (2–3), first forced use of a friend
  (1), Soften II gut-punch (1), Sanctuary relight template (1–2),
  endings: bad 2 / mediocre 2 / good 3 / true 3–4.

## Prompt template

> {STYLE_TAG}, wide establishing illustration, {scene_desc}, cinematic
> composition, painterly depth, characters small in frame

- Avoid more than two interacting characters in one frame (AI failure
  mode); if a beat needs a crowd, use silhouettes/statues in midground.
- Characters must match their bust identifiers (hair silhouette etc.);
  when drift is unavoidable, favor back/side views.
